# FlowFarm.jl

**Summary:** Flow Farm is a tool that enables the user to perform wind farm optimization.

**Authors:** Jared Thomas, Andrew P.J. Stanley

**Features:**

* AEP
* Wind Farm Layouts

**Tools:**

* Wake Deficit Models
* Wake Deflection Models
* Power Models
